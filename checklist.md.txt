# Чек-лист проверки формы авторизации VK

## Позитивные проверки
- [ ] Вход по валидному номеру телефона и паролю
- [ ] Вход по валидной почте и паролю
- [ ] Восстановление пароля через СМС-код

## Негативные проверки
- [ ] Попытка входа с пустыми полями
- [ ] Вход с неверным паролем
- [ ] Вход с заблокированным номером телефона
